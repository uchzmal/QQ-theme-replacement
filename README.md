# QQ主题替换

思路和资源 by [欲归来南枝](https://space.bilibili.com/1495315743)
脚本制作 by [zmal/梦断忆-](https://space.bilibili.com/554419861)

## 总纲
找到QQ系统级data数据目录（即Mt管理器提取安装包时的数据目录1）→
找到QQ主题缓存文件夹→
随便应用主题，将此主题作为准备替换的主题→
删除此主题二级/三级文件夹包括压缩包里面的文件，将下载的文件移动过去→
停止运行QQ→应用成功 重新打开即可
